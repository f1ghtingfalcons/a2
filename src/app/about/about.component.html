About

Roleman

Welcome to the LASP Role Manager aka Roleman. This application makes it easy to create new users in the internal LDAP (WEBIAM) and add them to various roles throughout the lab. Please contact an administrator if you are unsure how to use this application. You can log a ticket with the web support jira project or email the web support group (web.support@lasp.colorado.edu). Changes made here will be reflected in other systems. You will need to be an admin or a project lead in order to change group or user information.

How To

Creating a New LDAP User

Creating new users is done on the 'Create' page. Select the 'Create' page from the nav bar or use this link: Create New User
Use the input boxes to fill in the following information:
  • First Name
  • Last Name
  • Email
A username will be generated automatically based on the user's first and last name. However, you can change the username to your liking.
Clicking the arrow will queue the user to be added to LDAP, but the new user won't be commited to LDAP until the 'Register Users' button has been selected.
Checking the 'Invite Users Via Email' box will send an invitation email with links for the user to activate their account.

Add a User to a Group

You can add users to groups using the Groups page. Select the groups page with the navbar or use this link: Groups Page
Use the search box to find groups. You can select a group by clicking on it in the list
Once a group has been selected, it will show its most recent membership. Using the search box at the top of the page, you can queue new users to be added to the group. Users can also be removed from a group by clicking the 'x' next to their name.
To update groups, make sure you click the 'Floppy Disk / Save' icon after making your changes. Groups will be returned to the list on the left of the page.

Edit A User

Visiting a user's profile will allow you to change their information. Users List
The user's information will be displayed on the page. You can change a user's information by typing in the text boxes. Clicking the 'Save Changes' button will save the changes in LDAP.
A user's profile page can also be used to add a user to a group. Simply search for the group in the text box and click on it. It will automatically be saved in LDAP.
A user's account can be locked, unlocked or reset as well. Simply press the corresponding button. When resetting a user account, they will receive an email with a link to re-activate their account.
The profile page also includes handy information including the last time their password was reset and the associated PWM log. Object classes are LDAP properties and should be the same between profiles.
